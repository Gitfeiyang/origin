习近平眼中的科技创

2014年6月，习近平总书记在两院院士大会讲话中强调，我国科技发展的方向就是创新、创新、再创新。

2018年7月，习近平总书记在中央财经委员会第二次会议上强调，关键核心技术是国之重器，必须切实提高我国关键核心技术创新能力，把科技发展主动权牢牢掌握在自己手里，为我国发展提供有力科技保障。

2018年9月，习近平总书记在考察中国一重集团有限公司时说：“国际上单边主义、保护主义上升了，逼着我们靠自己，走自力更生的道路，这不是个坏事。福兮祸所伏，祸兮福所倚，中国最终还要靠自己。”

中国如何靠自己突破核心技术，实现科技创新？看视频，了解习近平总书记给出的路线图。（央视记者 吴璇 张鑫）
